## 1. 接口描述
接口请求域名: /blog/findPageBlogByHit
获取访问量最高的博客列表
请求方式: GET

## 2. 输入参数
| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
|  pageNum  |  是   | Number | 返回第几页的热门博客 |
| blogClass |  是   | 'hit'  |       博客类型       |

## 3. 输出参数
| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
|   list   | Array  |   热门博客的数组   |
| pageNum  | Number | 当前访问的是第几页 |
|  pages   | Number |    总共有多少页    |
|  total   | Number |  总共有多少条数据  |

## 4. 示例

/blog/findPageBlogByHit?pageNum=1&blogClass=hit
