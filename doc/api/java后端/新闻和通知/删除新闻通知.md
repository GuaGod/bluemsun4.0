## 1. 接口描述
接口请求域名: news/deleteNews
增加作品
请求方式: GET

## 2. 输入参数
| 参数名称  | 必选  |  类型  |         描述         |
| id | 是 | String | 作品id |
## 3. 输出参数
| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
|   stateCode   | String  |   状态码 1是成功   |
| message  | String | 描述信息 |

## 4. 示例

### 示例1 查询结果列表

#### 输入示例

news/deleteNews?id=68

#### 输出示例

{
    stateCode: "1",
    message: "删除成功",
}
